【函数式编程入门经典系列】第3章-高阶函数

Jul 28, 2018 | 函数式编程入门经典系列 | 阅读
  1. 高阶函数（Higher-Order Function，简称HOC）：接受另一个函数作为其参数的函数。

  1. JavaScript基础数据类型：（6+1）
    Number、String、Boolean、Object、null、undefined
    以及ES6新增的Symbol

  2. 高阶函数是接受函数作为参数并且/或者返回函数作为输出的函数。

  3. 高阶函数就是定义抽象。

  4. obj.hasOwnProperty():判断对象是否包含特定的自身（非继承）属性。

    const forEachObject = (obj,fn) => {
    for(var property in obj){
        if(obj.hasOwnProperty(property)){
            //以key和value作为参数调用fn
            fn(property,obj[property])
        }
    }
}

  5. 断言（assert）：值为true或false。

    const unless = (predicate,fn) => {
    if(!predicate){
        fn()
    }
}

forEach([1,2,3,4,5,6,7],(number) => {
    unless((number % 2), () => {
        console.log(number,"is even");
    })
})
