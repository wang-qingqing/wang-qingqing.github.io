在编写博客内容时，经常会用到一些hexo的命令。

  1. 生成静态页面命令

    hexo g                                                    
或           
hexo generate

  2. 部署文件

    hexo d                                                    
或       
hexo deploy

  3. 开启hexo服务器

    hexo s

  4. 在source/_posts文件夹下新建.md文件

    hexo new 'new MarkDown File under source/_posts'

  5. 在source文件夹下新建页面 

    hexo new page 'new page under source'
